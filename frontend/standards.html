This Project
My goal was to create a bulk search function specifically for civil engineering standards better than ctrl+f. This searches multiple documents at once and accepts naturalistic language inputs, so I think this succeeds.

This is technically "powered by AI". Like any good AI, it grows more powerful the more documents it gains. Help me by telling me about public standards YOU constantly search!
