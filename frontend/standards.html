This Project
This is a LLM powered database search that helps people find answers to civil engineering code questions. Questions are valid inputs, and you don't need exact phrasing to find what you need.

This project is powered by (local) AI! Your questions are private and not stored. It's trivial to add more files, so tell me the ones you use most and soon you can search them too!
